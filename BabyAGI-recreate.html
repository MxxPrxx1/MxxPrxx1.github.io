BabyAGI, the Start of Computers Wokring on Their Own

This project was inspired by and guided by Yohei Nakajima's BabyAGI github page and Jeffrey Quesnelle's blog post, both of which I recommend you check out.

This project is BabyAGI that doesn't use OAI's API. BabyAGI essentially is a script that uses an LLM to act as different roles like a task executer or task creator to pursue its goal. These different roles allow the overall agent breaks down its objective into smaller tasks and continually execute these tasks until it achieves its objective.
